# sk-server
scikit-learn server

## install environment

1. install [uv](https://docs.astral.sh/uv/)

2. install python

```shell
uv python install 3.12
```

3. init project

```shell
uv init
```

4. create venv

```shell
# pin python version
uv python pin 3.12
# create venv
uv venv
```

5. install python package

```shell
uv add scikit-learn
```
